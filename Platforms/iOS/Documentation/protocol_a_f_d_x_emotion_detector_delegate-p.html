- (void) detector: (AFDXEmotionDetector *)  detector
hasResults: (NSArray *)  emotions
forImage: (NSImage *)  image
atTime: (NSTimeInterval)  time 

Called for all frames captured. Contains information for emotions detected for a particular time.

When an emotion has been detected at a specific point in the video stream, this method will be called with an array of AFDXEmotionMetric objects. The number of objects passed equals to the number of emotions requested when the detector was initialized.

A reference to the actual image that was analyzed is passed as well. This can be useful for displaying in a view or doing other relevant processing.

The time into the stream that the emotion occurred is also available. This is a relative time composed indicating the number of seconds into the processing stream that the frame occurred.

This method will also be called if captured frame did not result in the detection of any emotions. This may happen for when no face is detected. In this case the array of AFDXEmotionMetric objects will be nil.

Parameters
detectorA pointer to the AFDXEmotionDetector object.
emotionsAn array of pointers to AFDXEmotion objects which represents the emotions that were expressed. Nil if none were detected for a particular frame.
imageA reference to the image which was analyzed.
timeThe time into the stream that the emotion was expressed.

- (void) detectorDidStartDetectingFace: (AFDXEmotionDetector *)  detector

Indicates that the face detector has started tracking a face.

When the face tracker detects a face, this method is called. The receiver should expect that tracking continues until a call to detectorDidStop: occurs.

Parameters
detectorA pointer to the AFDXEmotionDetector object.
- (void) detectorDidStopDetectingFace: (AFDXEmotionDetector *)  detector

Indicates that the face detector has stopped tracking a face.

When the face tracker no longer detects a face, this method is called. The receiver should expect that there is no face tracking until a call to detectorDidStart: occurs.

Parameters
